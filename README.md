# Порівняння методу Монте-Карло та аналітичних імовірностей  
## Кидання двох кубиків з значеннями від 1 до 6 (1 000 000 імітацій)

У цьому експерименті результати імітації методом Монте-Карло для кидання двох гральних кубиків були порівняні з теоретично обчисленими аналітичними імовірностями.

## Висновки

- **Результати методу Монте-Карло дуже близькі до аналітичних значень**, що підтверджує коректність реалізації симуляції та її статистичну збіжність при 1 000 000 імітацій.

- **Максимальна відносна похибка не перевищує 0.8 %**, що демонструє високу точність експерименту.

- **Найменша похибка спостерігається для суми 7**.

- **Більші відхилення характерні для крайніх значень (2 і 12)**, оскільки вони мають меншу ймовірність появи та потребують більшої кількості імітацій для ще точнішої оцінки.

### Загальний висновок
Метод Монте-Карло при достатній кількості імітацій забезпечує високу точність і узгоджується з аналітичними розрахунками.
